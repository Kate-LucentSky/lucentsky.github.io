介紹

Lucent Sky AVM 是一個幫助開發者及相關人員偵測程式碼弱點並自動修正的系統。Lucent Sky AVM 包括了 CLEAR Engine（分析及修正引擎）、 Web UI 及 Developer UIs 。其中 Developer UIs 包括：Lucent Sky AVM CLI、Lucent Sky AVM Visual Studio 擴充元件和 Eclipse 外掛元件。

Prerequisites for Developers

Developers should be familiar with the following tools:

  • C#, Java or VB.NET languages
    • This guide uses C# (which has a similar syntax with Java and VB.NET) in its examples and screenshots.
  • Visual Studio 2012 or Visual Studio 2013

Installing Visual Studio Extension

In this section, you will see:

  • A demonstration of how to install and configure Visual Studio Extension

At the end of this section, you will be able to:

  • Prepare Visual Studio to communicate with a Lucent Sky AVM Server
  • Install AVM Visual Studio Extension on a development machine

To prepare Visual Studio to communicate with AVM Server

  1. Close any Visual Studio instance that's running. Open the Visual Studio configuration file (usually located at C:\Program Files (x86)\Microsoft Visual Studio 11.0\Common7\IDE\devenv.exe.config, where 11.0 is the version of Visual Studio) with a text editor. Modify the system.serviceModel section to enable Visual Studio to communicate with a Lucent Sky AVM server.

To learn more about how to modify the configuration file, click the following article number to view the article in the Lucent Sky Knowledge Base:
    KB390837 Administration of Lucent Sky AVM Visual Studio Extension

To install AVM Visual Studio Extension

  1. Double-click on the AVM Visual Studio Extension setup file (CLEAR_VSIX.vsix).
  2. Follow the installation wizard to complete the setup.

To learn more about the installation and configuration of AVM Visual Studio Extension, as well as system requirements, click the following article number to view the article in the Lucent Sky Knowledge Base:
    KB390837 Administration of Lucent Sky AVM Visual Studio Extension

Using Visual Studio Extension

In this section, you will see:

  • A sample procedure for analyzing and mitigating the vulnerabilities in an application with the AVM Visual Studio Extension

At the end of this section, you will be able to:

  • Scan an application for vulnerabilities and automatically fixing them using Lucent Sky AVM Visual Studio Extension
  • Review the identified vulnerabilities and apply their Instant Fix using Lucent Sky AVM Visual Studio Extension

To analyze and mitigate the vulnerabilities in an application using Lucent Sky AVM AVM Visual Studio Extension

  1. For web applications, open the solution containing the project of the application in Visual Studio. For web sites, open the web site in Visual Studio.
  2. Under Solution Explorer, right-click on the project or website to be analyzed, and then select Analyze Security Issues.
  3. The Analyze Security Issues dialog will pop up. Click Scan to start the analysis. You can also optionally change the analysis and mitigation parameters for each vulnerability categories and source vectors.
  4. The Analysis Results window will appear. When the analysis is in progress, the window will refresh itself to show the current stage. You can also click Refresh at its upper-right corner to manually refresh the window. There are 6 stages:
    • Building
    • Compiling
    • Analyzing (S-1)
    • Analyzing (S-2)
    • Analyzing (S-3)
    • Analyzing (S-4)
  5. Once the analysis is completed, the vulnerabilities will be shown in the Analysis Results window. Expanding a vulnerability category will show all vulnerabilities under that category. Expanding a vulnerability will show the Statements and Instant Fix (if available) of that vulnerability. Double-click one of the Statements or Instant Fixes to go to the line of code in question.
  6. To mitigate a vulnerability, right-click on its Instant Fix and click Mitigate. The Instant Fix will be automatically applied in Visual Studio.
